这种题目都不难，只要抓住面试官想考察的重点即可，就这道题来说，不必理会具体的概率，只要p(x)调用g(x)获取到4或者5，就再次调g(x)重试，直到输出结果是1、2、3，伪代码如下：
```java
int a(x)  {
    int x = g(x);
    whiile (x > 3) {
        x = g(x)
    }
    return x;
}
```
